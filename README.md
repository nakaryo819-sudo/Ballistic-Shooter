# Ballistic Shooter (弾道予測＆エイムトレーニング)

Unityで制作した、物理演算とゲームループを組み合わせた3Dシューティングゲームです。
「物理的な弾道計算」と「動的な難易度調整」を実装し、ゲームとしての完成度を高めました。

## 🎮 実装済みの機能
* **物理シューティング**: 重力の影響を受ける弾丸を発射し、物理衝突によってターゲットを破壊します。
* **動的な難易度上昇**: プレイ時間やスコアに応じて、ターゲットの消失速度が徐々に上がり、緊張感を生み出します。
* **ゲームサイクルの構築**:
    * スコア管理システム（UI更新）
    * 終了条件（ゲームオーバー判定）
    * シーンのリセット機能
* **弾道の可視化**: TrailRendererによる軌跡の描画。

## 🛠 技術的なこだわり
* **難易度曲線**: 単調にならないよう、Time.deltaTimeを用いてターゲットの寿命を動的に制御しています。
* **状態管理**: ゲーム中、ゲームオーバー時、リセット時のステート管理を実装しました。

## 🛠 使用技術
* Unity 2022.3
* C#
* uGUI (スコア表示、ゲームオーバー画面)
