# Ballistic Shooter (弾道予測シューティング)

Unityで制作した、物理演算を活用したシューティングゲームです。
機械工学で学んだ物理法則（運動方程式）をプログラムに落とし込み、弾道をリアルタイムに計算・予測する機能を実装しました。

## 🎮 こだわりポイント
* **物理演算の自力実装**: UnityのRigidbodyに頼りすぎず、重力と初速から着弾地点を計算するロジックを組んでいます。
* **予測線の描画**: プレイヤーが直感的に狙えるよう、放物線の軌道をLineRendererで可視化しました。

## 🛠 使用技術
* Unity 2022.3
* C#
* Visual Studio
